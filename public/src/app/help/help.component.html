How to use:

The purpose of this program is to be able to combine several waves together. When you press a key on your keyboard that corresponds to a musical note (CDEFGAB), the base frequency is automatically set to the frequency of that musical note. Additional waves you add will be based on the base frequency wave. For example, if you add a second wave (by pressing the "New Wave" button") and set the frequency to 20 Hz, the new wave will have a frequency of 20 Hz ABOVE the base frequency. The amplitude works in much the same way, except that it is described as a percentage of the base wave's amplitude (instead of being added). Now, when you press a key, the two waves will be combined together to form a harmonic.

How do I play sharps and flats?

Press shift + key to make a sharp, and alt + key to make a flat. For example to play a C#, press shift + C.

How was this built?

This was built from scratch using Typescript, a programming language that is transpiled to Javascript (ECMAScript 5). In addition to Typescript, a framework called Angular 4 was utilized. This framework combines common directives and components to make this app easier to build
